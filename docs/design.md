## コンセプト

パチスロの「波」構造（通常時の我慢 → ボーナスの爆発）に、テリーのワンダーランド式のモンスター配合を埋め込む。
**育成がスロットの期待値を実質的に引き上げる**のが核：リールの確率テーブルは固定で公正なまま、
育成努力が「バトルラッシュ突入後のリターン」を伸ばす。パチスロの設定6を自分の手で作る感覚。

参考作品: Luck be a Landlord（スロット×成長）、テリーのワンダーランド（配合）、パチスロのAT機（波設計）

## スロット仕様

### 方向性
カジュアル役構成＋確率明示＋演出全振り。確率テーブルはゲーム内で常時閲覧可能（「確率を大事に」の見える化）。

### 操作（目押し・オート両立の設計）
実機パチスロの「取りこぼし」方式を採用。**当選抽選はスピン開始時に確定**（公正・タイミング非依存）、
目押しは「当選した報酬を満額取れるか」にだけ影響する。

- **基本**: タップ/ボタンで3リールを順に止める。2リール目まで揃うとリーチの間が生まれる
- **目押し要素**: フラッシュ図柄（レア素材・ボーナスコイン）当選時、リール上で図柄が光る。
  タイミング押し成功 → 満額獲得、失敗 → 半額（取りこぼし）。当選自体は消えない
- **オートスピン**: 序盤の階層クリアで解放。目押しボーナスは自動では取れない（満額不可）
  → 「手で打つと期待値が少し上がる」スキルとの共存。放置勢はオートで回せる

### 図柄・役テーブル（初期ドラフト、要バランス調整）

| 図柄 | 役 | 確率(目安) | 効果 |
|---|---|---|---|
| 銅コイン | 3つ揃い | 1/8 | ベット×2 払い出し |
| 銀コイン | 3つ揃い | 1/20 | ベット×5 |
| 金コイン | 3つ揃い | 1/50 | ベット×15 |
| タマゴ | 3つ揃い | 1/40 | モンスターのタマゴ獲得 |
| 剣/ハート/星 | 通常時は小役 | 各1/15 | 少額＋ラッシュ中はモンスターブースト |
| 扉（バトル図柄） | 3つ揃い | 1/25 | バトルラッシュ突入 |
| フラッシュ図柄 | 目押し役 | 1/30 | 目押し成功で満額（素材/コイン） |

- 通常時RTP目標: 約90〜95%（じわじわ減る → ラッシュで取り返す波を作る）
- **天井**: バトル非突入が40スピン続くと突入確定。残りスピン数はメーター表示
  （ハマリの絶望を「天井接近の期待」に変換する）

### 演出（ドーパミン設計の優先順位）
1. リーチ（2リール目一致で3リール目スロー＋煽りSE）
2. ラッシュ突入カットイン（画面フラッシュ＋モンスター登場演出）
3. 連続揃い時のコンボ表示・払い出しコインの物理シャワー
4. 敗北間際からの復活演出（低確率で発生、期待値には織り込み済み）
5. 画面振動・ヒットストップ・音の3点セットは全報酬イベントに

## バトルラッシュ仕様

- 育成パーティ3体 vs 敵グループ、完全自動戦闘（観戦が気持ちいい見せ方を優先）
- ラッシュ中もスピン継続。揃った役でブースト:
  - 剣3つ → 全員追撃
  - ハート3つ → 全員回復
  - 星3つ → 先頭モンスターの継承スキル（必殺技）発動
- 勝利: 大量コイン＋タマゴ（高階層ほどレア系統）＋配合素材
- 敗北: 投入コインの一部＋素材は持ち帰り（保険）
- ステータス: HP / ATK / DEF / SPD ＋ 継承スキル1個

## モンスター・配合仕様

### 系統（4系統）
獣 / ドラゴン / スライム / アンデッド

### 配合ルール
- 2体を配合 → 系統の組み合わせで新種が決まる（組み合わせ表）
- 親のスキルから1個選んで継承
- 配合後、親は消滅（テリワン準拠）。子はレベル1だが基礎ステが親の分だけ上乗せ

### 種族数（初期15種）
- ベース4種（各系統1種、タマゴから孵化）
- 配合で10種（同系統×同系統4通り＋異系統6通り）
- 隠し1種（特定のレア組み合わせ＋条件。図鑑にシルエット表示で匂わせる）

## 進行構造

- 階層登りエンドレス。バトルラッシュ勝利で階層+1、敵が強化
- 10階層ごとにボス（タマゴ確定ドロップ、レア系統解放）
- ハイスコア＝到達階層。localStorage セーブで何日も育成継続可能

## 画面構成

1. **メイン画面**: スロット中央、パーティ3体表示（待機モーション）、コイン/階層/天井メーター
2. **育成・配合画面**: 手持ちモンスター一覧、配合UI、餌やり（素材消費で強化）
3. **図鑑**: 発見済み種族＋シルエット（コンプ欲）
4. **確率表**: 役テーブル常時公開
